TrackAttack

Solo developer • 2025
Full-stack Web App React Node.js Express MySQL Socket.IO

TrackAttack is a full-stack web platform that allows sim racers to manually log lap times, attach video proof and compare performance across tracks and drivers.The project focuses on structured data modeling, real-time interaction, and scalable architecture decisions.

Overview

Problem

Sim racers often struggle to track and analyze their lap times across different tracks and compare their performance with other drivers. Existing solutions are either too basic, lack real-time features, or are not tailored to the specific needs of the sim racing community.

Solution

TrackAttack provides a comprehensive platform for sim racers to log their lap times, attach video evidence, and compare their performance with others. The app features a structured data model for efficient storage and retrieval of lap data, real-time updates using Socket.IO, and a user-friendly interface built with React.

Focus

The main focus of this project was to build a scalable and maintainable full-stack web application. I prioritized clean architecture, efficient data modeling, and a responsive user interface that enhances the sim racing experience.
